OFICINA 2      

Título da Oficina: Shell Script, RRDToll, SNMP e Linux para monitorar equipamentos de rede.
Ministrante: Clébio Dossa - UPF

Resumo:
Uma rede de computadores, indiferente do porte, necessita ser gerenciada e monitorada para manter o funcionamento e prever anomalias. Cada ambiente de rede tem suas particularidades, assim, nem sempre uma única ferramenta se mostra eficiente para cumprir as tarefas de monitoramento. Uma bom método de análise de dados se faz com o uso de gráficos em linha de tempo. Para isso esta oficina tem como objetivo demonstrar métodos de coleta de dados, armazenamento e análise usando o protocolo SNMP, juntamente com formas de desenvolver scripts para coleta remota de dados e a maneira de gerar gráficos com o RRDTool.
